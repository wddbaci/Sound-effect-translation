# WavTranslator - AI音频批量翻译工具

WavTranslator 是一个基于 DeepSeek API 的桌面工具，可以批量将音频文件名中的英文部分翻译成中文，并按照用户指定的命名模式重命名文件。支持多种常见音频格式，内置翻译缓存以避免重复请求，适合音效库、音乐文件整理等场景。

## ✨ 功能特性

- 🔍 **自动提取文件名**：将文件名中的英文部分（驼峰命名自动分词）提取出来进行翻译
- 🌐 **多格式支持**：支持 `.wav`、`.mp3`、`.flac`、`.aac`、`.m4a`、`.ogg`、`.wma`、`.aiff`、`.ape`、`.opus` 等常见音频格式
- ⚡ **批量高效翻译**：使用 DeepSeek API，支持并发批次翻译，自动处理限流重试
- 💾 **本地缓存**：翻译结果自动缓存，重复文件无需二次请求，节省 API 调用
- 🖥️ **图形界面**：基于 Tkinter 的简单易用界面，实时显示日志和进度
- 🔐 **安全配置**：API Key 保存在本地文件，并设置权限为仅当前用户可读写
- 📁 **多种命名模式**：支持 `中文_英文`、`只中文`、`英文_中文` 三种命名方式

## 📦 安装

### 环境要求
- Python 3.6 或更高版本
- 操作系统：Windows / macOS / Linux（需要支持 Tkinter）

运行程序

bash
python WavTranslator.py
设置 API Key

在 DeepSeek 官网 platform.deepseek.com 注册并获取 API Key。
将 API Key 粘贴到输入框，点击「保存 Key」（保存后 Key 会加密存储在本地）。
选择文件夹

点击「浏览...」选择包含音频文件的文件夹。
选择命名模式

支持三种模式：

中文_英文：原文件名前加上中文（如 脚步声_Footstep.wav）
只中文：仅保留中文翻译（如 脚步声.wav）
英文_中文：原文件名后加上中文（如 Footstep_脚步声.wav）
开始翻译

点击「开始翻译重命名」，程序会自动提取文件名中的英文、调用 API 翻译、并重命名文件。
翻译过程中可随时点击「取消」停止操作。
查看日志与进度

日志区域会显示详细步骤，进度条和计数实时更新。
⚙️ 配置与缓存

API Key 存储位置：

Windows：%APPDATA%\WavTranslator\config.json
macOS/Linux：~/.config/WavTranslator/config.json
文件权限设置为仅当前用户可读写。
翻译缓存位置：

同上目录下的 translation_cache.json，自动保存，避免重复翻译。
清空缓存：

界面中点击「清空缓存」按钮可删除所有缓存记录，下次将重新翻译。
📝 注意事项

翻译质量取决于 DeepSeek API，请确保网络畅通且 API Key 有效。
批量翻译时会根据 CONCURRENT_BATCHES 并发请求，默认 2 批同时进行，避免触发限流。
文件名中的英文部分会先进行驼峰分词处理（例如 FootstepOnGrass 会被拆分为 Footstep On Grass），提高翻译准确性。
重命名时会自动处理文件名冲突（添加数字后缀）。
